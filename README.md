# Perfil DISC - Prisma Comportamental PRO

Site responsivo para análise de perfis comportamentais DISC (Dominância, Influência, Estabilidade, Conformidade).

## Como usar

Acesse o `index.html` em qualquer navegador ou faça upload para o GitHub Pages.

## Publicar no GitHub Pages

```bash
# 1. Crie um repositório no GitHub
# 2. Execute:
git init
git add .
git commit -m "Initial commit - DISC behavioral analysis"
git branch -M main
git remote add origin https://github.com/SEU_USUARIO/SEU_REPO.git
git push -u origin main

# 3. No GitHub, vá em Settings > Pages > branch: main, folder: / (root)
```

## Estrutura

```
├── index.html          # Página principal
├── style.css           # Estilos responsivos
├── app.js              # Lógica do frontend
├── perfis.json         # Dados dos perfis comportamentais
├── itens.json          # Itens comportamentais por contexto
└── zona_cinzenta.json  # Interpretação da zona cinzenta
```
